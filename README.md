# AI Chatbot Assistant

An intelligent NLP-based chatbot built using PyTorch and NLTK for intent classification and automated responses.

---

## Features

- NLP preprocessing
- Intent classification
- Deep Neural Network
- Bag-of-Words vectorization
- Model save/load functionality
- Function mapping support

---

## Tech Stack

- Python
- PyTorch
- NLTK
- NumPy

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Chatbot-Assistant.git
cd AI-Chatbot-Assistant
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run the Project

```bash
python chat.py
```

---

## Example

```bash
Enter your message: hello
Bot: Hi there!
```
